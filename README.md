  # Ambient Intelligence Lab's Website

  Visit **[ambient-intelligence-lab-umich-eecs.github.io/ami_lab_website](https://ambient-intelligence-lab-umich-eecs.github.io/ami_lab_website)** 🚀

  ## Local preview

  To preview the site locally with the same base path used on GitHub Pages, run:

  ```bash
  bundle exec jekyll serve --config _config.yaml,_config.local.yaml
  ```

  Then open:

  ```text
  http://127.0.0.1:4000/ami_lab_website/
  ```
